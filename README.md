## Play on web
https://gamejam-2023-web-0.vercel.app/

## Install and run
- Install [Node.js](https://nodejs.org/en)

build and run
```sh
npm install
npm run build
npm start
```

build and run with NODE_ENV
```sh
NODE_ENV=production npm install
NODE_ENV=production npm run build
NODE_ENV=production npm start
```
